REFERRAL SYSTEM:
- Each customer gets a unique referral code (e.g. GDH-PRIYA-4821)
- They share this code with friends
- Friend enters referral code during checkout
- Both get points
- Dashboard shows: "Aapne X logon ko refer kiya"
- 5 referrals milestone: unlock special reward notification

REDEEM POINTS:
- 100 points = ₹10 discount on next order
- Minimum 200 points needed to redeem
- Redeem modal: slider to choose how many points to use
- Shows: "Aap ₹X ki choot payenge"
- Owner gets notified via WhatsApp with redemption details
